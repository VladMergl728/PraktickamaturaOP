Fedora Linux Server - správa z Windows 11

1. Síť + statická IP

Zobrazení rozhraní: ip a Nastavení IP: nmcli con show Úprava: sudo nmcli con mod "Wired connection 1" ipv4.addresses 172.16.0.254/16 sudo nmcli con mod "Wired connection 1" ipv4.gateway 172.16.0.1 sudo nmcli con mod "Wired connection 1" ipv4.dns 8.8.8.8 sudo nmcli con mod "Wired connection 1" ipv4.method manual Restart: sudo nmcli con up "Wired connection 1"

2. SSH (bez hesla)

Na Windows: ssh-keygen Odeslání klíče: ssh-copy-id user@172.16.0.254 Test: ssh user@172.16.0.254

3. KDE (GUI)

sudo dnf groupinstall "KDE Plasma Workspaces" -y sudo systemctl set-default graphical.target

4. VNC server

sudo dnf install tigervnc-server -y vncserver Na Windows použij VNC Viewer

5. Firewall (firewalld)

Zapnutí: sudo systemctl enable --now firewalld Povolení portů: sudo firewall-cmd --permanent --add-service=ssh sudo firewall-cmd --permanent --add-port=5901/tcp sudo firewall-cmd --permanent --add-port=8080/tcp sudo firewall-cmd --permanent --add-port=8081/tcp sudo firewall-cmd --permanent --add-port=9090/tcp sudo firewall-cmd --permanent --add-service=samba Reload: sudo firewall-cmd --reload

6. Cockpit

sudo dnf install cockpit -y sudo systemctl enable --now cockpit Web: http://172.16.0.254:9090

7. Samba (sdílení)

sudo dnf install samba -y Edit: sudo nano /etc/samba/smb.conf Přidat: [SHARE]
path = /home/user/share
browsable = yes
writable = yes
guest ok = yes Spuštění: sudo systemctl enable --now smb

8. Web server (Apache)

sudo dnf install httpd -y Spuštění: sudo systemctl enable --now httpd Porty: sudo nano /etc/httpd/conf/httpd.conf Přidat: Listen 8080
Listen 8081 Restart: sudo systemctl restart httpd

9. SCP / SFTP test

Na Windows: scp index.html user@172.16.0.254:/var/www/html

10. Backup script

nano backup.sh Script: #!/bin/bash
tar -czf /backup/backup_$(date +%F).tar.gz /home/user/share Práva: chmod +x backup.sh Spuštění: ./backup.sh

11. Cron (automatizace)

crontab -e Denně: 0 2 * * * /home/user/backup.sh
