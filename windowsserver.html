Windows Server z Linuxu - Krok za krokem

1. Síť (NAT + IP)

Klikni: Nastavení VM → Network → NAT
Windows Server:
  • IP: 192.168.1.10
  • Maska: 255.255.255.0
  • Gateway: 192.168.1.1
  • DNS: 192.168.1.10

2. DHCP + DNS (Windows Server)

Klikni: Server Manager → Add Roles → DHCP + DNS
DHCP Setup:
  • Scope: 192.168.1.100 - 192.168.1.200
Rezervace:
  • Klikni: DHCP → Reservations → New
  • Zadej MAC adresu Linuxu

3. IIS (Web server)

Klikni: Add Roles → IIS
Přidání webu:
  • Server Manager → IIS Manager
  • Sites → Add Website
  • Port: 8080
  • Port: 8081 (druhý web)

4. FTP Server

Klikni: IIS → Add FTP Site
Nastav složku (např. C:\inetpub\wwwroot)
Povolit login uživatele

5. Firewall

Klikni: Windows Firewall → Advanced settings
Povol:
  • RDP (3389)
  • FTP (21)
  • HTTP (8080,8081)
  • DNS (53)
  • DHCP (67,68)

6. Storage Spaces

Klikni: Server Manager → File and Storage Services
Storage Pools → New Pool
Přidej disky → vytvoř Volume

7. Linux (KDE + nástroje)

Aktualizace: sudo apt update && sudo apt upgrade KDE: sudo apt install kde-plasma-desktop RDP klient: sudo apt install remmina FTP klient: sudo apt install filezilla

8. RDP připojení

Spusť Remmina → New Connection
IP: 192.168.1.10
User: vytvořený admin

9. FTP test

Spusť FileZilla:
  • Host: 192.168.1.10
  • Port: 21
Nahraj: index.html

10. Ověření

Otevři v prohlížeči:
  • http://192.168.1.10:8080
  • http://192.168.1.10:8081
